# Análise de Vendas com Excel

Este projeto tem como objetivo realizar uma análise exploratória de dados de vendas utilizando o Microsoft Excel, com foco na criação de métricas, tabelas dinâmicas e visualizações para apoio à tomada de decisão.

##  Descrição do Projeto

A análise foi desenvolvida a partir de um conjunto de dados fictício de vendas, contendo informações sobre produtos, categorias, datas, quantidades vendidas e preços unitários.  
O projeto simula um cenário real de análise de desempenho comercial.

##  Estrutura do Arquivo

O arquivo Excel está organizado da seguinte forma:

- **database**: base de dados original utilizada para as análises  
- **tabelas_dinamicas**: tabelas dinâmicas criadas a partir da base de dados  
- **graficos**: gráficos gerados com base nas tabelas dinâmicas  

##  Análises Realizadas

- Cálculo de faturamento a partir da quantidade vendida e preço unitário  
- Análise de faturamento por produto e por categoria  
- Análise da quantidade vendida por produto  
- Análise de vendas ao longo do tempo  
- Criação de gráficos para visualização dos principais indicadores  

##  Ferramentas Utilizadas

- Microsoft Excel  
- Tabelas Dinâmicas  
- Gráficos  

##  Objetivo

Demonstrar habilidades práticas em análise de dados utilizando Excel, incluindo organização de dados, criação de métricas, análise exploratória e visualização de informações.
